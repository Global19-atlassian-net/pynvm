Welcome to pynvm documentation!
==================================

This framework aims to bring newer NVM (*non-volatile memory*)/SCM (*storage-class memory*)
technology functionalty to Python ecosystem. This project contains mainly the Python bindings
for some of the libraries present in the the excellent `NVM Library <https://github.com/pmem/nvml>`_
together with modifications to make it Pythonic and easy to use without modifying the Python
interpreter itself.
